## Usage

This is a vite project with typescript and react. You can use `npm run dev` to start a dev server or `npm run build` for a full build.

It is using vitest for testing. You can use `npm run test` to run the test suite.

## Contents

This uses React Router for a SPA. It has 2 main pages, /login and /account.

It uses Apollo to connect to the GraphQL endpoint and i18next for the localization.

It currently has 2 languages, english and spanish.

It uses code generation to generate the types of the GraphQL queries user in `/src/queries`.
